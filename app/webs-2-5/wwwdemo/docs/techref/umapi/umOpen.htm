umOpenGoAhead WebServer APIGoAhead EMF

Synopsis

Initializes the User Management module.

Prototype

 #include "um.h"

 int umOpen();

Parameters

None.

Description

umOpen() must be called before accessing User Management functions.

Return Value

umOpen returns 0 if successful or -1 on error.

Example

int nCheck = umOpen();

See Also

umClose(), umRestore()
