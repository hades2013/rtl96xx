WebServer 2.5 Overview

GoAhead WebServer 2.5 is a fully-featured embedded Web server that leverages Active Server Pages, embedded JavaScript, and in-process CGI forms processing (GoForms™) to deliver highly efficient and effective dynamic Web page creation. This release adds built-in support for MatrixSSL and an enhanced URL parser that greatly increase security.

Main Features

  • Active Server Pages
  • Embedded JavaScript
  • Standard CGI Implementation
  • GoForms™ (in-memory CGI processing)
  • MatrixSSL™
  • URL Handlers
  • Extensive API Documentation
  • Small footprint -- 50K RAM
  • Fast -- over 50 requests per second
  • Fully standards compliant
  • Source code eval

System Requirements

  • TCP/IP stack
  • Time/date
  • File system optional

Operating System Suppport

  • eCos
  • LINUX
  • LynxOS
  • QNX
  • VxWorks 5.3.1
  • VxWorks 5.4
  • Windows CE
  • Windows 95 and above
  • Windows NT and above
