The following is description of onu rds directory structure:

./bridge      - Bridge related files
./build       - Output files
./cli         - cli related files
./dvrs        - Driver files
./linux       - Linux module file
./multicast   - Multicast related files
./oamp        - oamp related files
./port        - Port related files
./qos         - QoS related files
./rstp        - RSTP related files
./snmp        - SNMP related files
./sys         - System management related files
./vos         - Virtual operating system files
./vxworks     - vxWorks project files


To build onu in LINUX:
- open your source code directory
- ./make

To build onu in vxWorks:
- Open Tornado IDE
- ...
